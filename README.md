# Murali Krishna Perneedi Portfolio

A modern React + Tailwind CSS portfolio for Murali Krishna Perneedi, focused on frontend engineering experience across Angular, Next.js, React, TypeScript, healthcare web applications, reusable UI systems, and performance optimization.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in your terminal.

## Production Build

```bash
npm run build
npm run preview
```

## Deploy

Deploy the generated `dist` folder to Vercel, Netlify, GitHub Pages, or any static hosting provider.
